# AXI4 Stream frivers for Digilent ZMOD

## ZMOD DAC

### AXIS Data format

|0 [31:30] | data I [29:16] | 0 [15:14] | data Q [13:0]|
|-|-|-|-|

### Gain adjust

Output gain for the ZMOD DAC is configured through the output FSADJI and FSADJQ.
